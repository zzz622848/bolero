    • Datasets¶

    BOLeRo provides datasets, e.g. for imitation learning.

    The following table gives an overview of the datasets that are provided by BOLeRo.

    Dataset name Usecase
    Minimum jerk Minimum jerk trajectory

    Minimum Jerk¶

    make_minimum_jerk() provides a trajectory that minimizes the jerk, i.e. the 3rd derivative of the position with respect to time.
